## 🛠️ 安装与运行说明

### Windows 用户
1. 下载 `[软件名]-setup.exe` 安装包。
2. 双击运行，按照提示安装即可。
3. *注意：由于未购买微软数字签名，部分杀毒软件可能会误报，请选择“仍要运行”。*

### macOS 用户
1. 下载 `[软件名].dmg` 镜像文件。
2. 双击打开，将软件图标拖动到 `Applications` (应用程序) 文件夹中。
3. 首次打开若提示“无法验证开发者”，请前往 `系统设置 -> 隐私与安全性 -> 仍要打开`。
